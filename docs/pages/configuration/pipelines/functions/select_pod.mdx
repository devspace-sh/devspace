This function is used to select the pod (an object that is a single instance of a running process in your cluster).


## Argumentss

### `--image-selector`

The image selector to be used when selecting a container.

### `--label-selector`

The lable selector to be used when selecting a container.

### `--container`

The container to be used.

### `--namespace`

The namespace to be used for the container.

### `--disable-wait`

If enabled, the pod will not wait for the container to be ready.

### `--timeout`

The amount of time the pod should wait. The default is 5 minutes.
