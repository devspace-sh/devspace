# Ensure Pull Secrets

This function deploys pull secrets in a configuration for a pipeline. PullSecrets are image pull secrets that will be created by devspace in the target namespace during devspace dev or devspace deploy.

## Arguments

### `--set`
Sets the configuration.

### `--set-string`
Sets the configuration as a string.

### `--from`
Reuses an existing configuration.

### `--from-file`
Reuses an existing configuration from a file.

## Flags
By default, flags are disabled. If a flag is enabled, the following actions are taken.

### `-all`
If enabled, ensures all pull secrets are deployed to a configuration.
