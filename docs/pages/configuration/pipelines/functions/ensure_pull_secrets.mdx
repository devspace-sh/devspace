This function deploys pull secrets in a configuration for a pipeline.

## Arguments

### Set

Sets the configuration.

### SetString

Sets the configuration as a string.

### From

Reuses an existing configuration.

### FromFile

Reuses an existing configuration from a file.

### All

If set to "true," ensures all pull secrets are deployed to a configuration.
